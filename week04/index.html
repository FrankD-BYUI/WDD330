JavaScript: Novice to Ninja Chapters 8, 12, 15

Lessons Learned:

  • addEventListener() can be used to add events to a form. For example, running some code when a form field is clicked on.
  • preventDefault() can be used to prevent a form from performing it’s default behavior.
  • JavaScript validation allows for better customizability and compatibility over HTML5 validation.
  • JS is prototype-based, which means instead of objects being instances of classes, they’re based on a prototype object.
  • Constructor functions remind me of functions in Python.
  • ES6 class declarations are even more similar to class-based languages and is preferred.
  • Static methods are called by the class rather than by an instance of the class and are not available to instances.
  • Properties are public by default but getters and setters can be used to limit access to them.
  • Object.create() is used to create a new instance of an object based on another object as the template.
  • It is considered good practice in JavaScript to limit inheritance chains to 2 classes deep.

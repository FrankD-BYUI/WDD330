HTML5 & CSS3 for the Real World Chapters 8 & 12

Notes:

  • Transforms are a way to move or otherwise manipulate how an element shows up on the page using CSS.
  • Translate is used to move an element. Translating an element does not affect the position of anything else on the page.
  • Scale is a way to change the sizing of an element on the page.
  • Rotate allows you to rotate an element.
  • Transforms are applied in the order they are listing. Changing the order changes the final result.
  • Transition is a way to have an element gradually change form one style to another
  • Animations are like transitions, but you can set them up with keyframes.
  • Canvas allows you to create a space to draw on (like a real life canvas)
  • Canvas drawing can be exported as images.
  • Once you finish creating a canvas object, it can no longer be modified.
  • SVG is a vector based image format that uses XML.
  • SVG can be easily edited and manipulated by CSS and JavaScript

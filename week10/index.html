MDN: Validating Forms and Using Fetch

Notes:

  • The form validation tools built into HTML5 allow for a wide variety of checks.
  • The Constraint Validation API allows JavaScript to interact with the built in HTML validation allowig for custom error messages and other events.
  • Custom validation can be built up using only JavaScript. Significantly more work, but far more customizable.
  • Fetch provides a convenient way to request resources over the internet using JavaScript.
  • Fetch requires a URL but can also include things such as credentials or data (in the form of JSON or even files).

Week 9: The Window Object and HTML5 APIs

Team Activity

Drumkit Activity

Javascript: Novice to Ninja Chapters 9 & 14

Notes:

  • The window object is the next level outside of the document object
  • All global variables are properties of the window object
  • alert(), confirm(), and prompt() are all methods of the window object
  • Window.location contains information on the documents URL
  • Window.history can be used to navigate in the browser's history
  • Cookies can be used to store name-value pairs for later use
  • The timing functions can be used to trigger events based on a timer
  • Animations use JavaScript to move elements on the page allowing the browser to set the timing
  • The data- attribute allows you to set customized attributes
  • Geolocation provides information on where the user is in the real world
  • Web Workers allow for something similar to multi-threading, allowing a script to move on while a difficult task continues to proccess in the background
  • Web-sockets allow for two way communications between the client and server
  • The Notification API allows the webpage to push notifications to the device
  • The Multimedia API allows for the control of embeded video and audio

Notes:

  • Properties can be added to functions in much the same way they can be added to objects.
  • Memoization can be used to cache the results of commonly calculated values.
  • IIFE: Immediately Invoked Function Expressions. A convinient way to create temporary variables and limit variable scope. Can be useful for initializing code.
  • Functions can redefine/rewrite themselves. Useful to have one function that can check browser compatability, then rewrite iteself to only have the code needed for that specific setup.
  • Recursive functions are able to call themselves. Useful when a complicated task can be broken down into repeated simple tasks.
  • Callbacks allow the program to set a function cal asside and come back to it later.
  • Ajax: Asynchronous JavaScript XML (although JSON files are used more than XML these days
  • Fetch API is now used to implement Ajax
